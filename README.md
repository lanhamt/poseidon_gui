# poseidon_gui
enjoy :)

## installation
```
$ git clone https://github.com/lanhamt/poseidon_gui
$ cd poseidon_gui
$ go run poseidon.go
```

## configuring gopath 
```
$ export PATH=$PATH:/usr/local/opt/go/libexec/bin
$ export GOPATH=$HOME/poseidon_gui  # or path from home to poseidon_gui
$ export PATH=$PATH:$GOPATH/bin 
```
